题目3：找出数组中重复的数字

在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

例如：[2, 3, 1, 0, 2, 5, 3]，输出：2 或 3。

思路一

将数组进行一次遍历并排序，由于题目上指明了所有数字都在0~n-1范围内，每次循环前，首先将下标i 与 对应的值nums[i]进行比较，相等则接着扫描下一个数字，不相等则再将nums[i]与 nums[nums[i]]比较，相等就找到了一个重复数字，不相等则交换位置。

例如：[2, 3, 1, 0, 2, 5, 3] ，第一个数2与对应下标0不相等，与下标2对应数字0也不相等，所以第一次循环后交换位置为[0, 3, 1, 2, 2, 5, 3]，由于第一个数0与下标0相等，则进入下一次循环：[0, 2, 1, 3, 2, 5, 3]->[0, 1, 2, 3, 2, 5, 3]……在第五次循环中，nums[4]=2与nums[nums[4]]=nums[2]=2相等，返回第一个相等的数字 2……在第七次循环中，nums[6]=3与nums[nums[6]]=nums[3]=3相等，返回第二个相等的数字 3。

代码实现：

class Solution {
    public int findRepeatNumber(int[] nums) {
        int temp;
        for(int i = 0;i<nums.length;i++){
            while(nums[i] != i ){
                if(nums[i] == nums[nums[i]]){
                    return nums[i];
                }
                temp = nums[i];
                nums[i] = nums[temp];
                nums[temp] = temp;
            }
        }
        return -1;
    }
}

题目4：二维数组中的查找

在一个 n * m 的二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

例如：如下二维数组，如果查找 7，返回true；如果查找20，返回false

1   2   8   9
2   4   9   12
4   7   10  13
6   8   11  15

思路一

查找某一个数字时，首先将该数字与右上角的数字比较，相等就直接返回true；如果小于右上角的数字，则排除了最后一列上的所有数字（因为右上角的数字是最后一列上最小的）；如果大于右上角的数字，则排除了第一行的所有数字（因为右上角的数字是第一列上最大的），这样每次查找都会排除一列或一行的数字，直到查找返回为空。

例如查找数字7，首先与右上角的9进行比较，9大于7，排除最后一列，右上角的数字变为8；再与8比较，8大于7，排除8所在的列，右上角的数字变为2；再与2比较，2小于7，排除2所在的那一行，右上角的数字变为4；再与4比较，4小于7，排除4所在的那一行，右上角的数字变为7；再与7比较时，相等直接返回true。

代码实现：

class Solution {
    public boolean findNumberIn2DArray(int[][] matrix, int target) {
        if (matrix == null || matrix.length == 0 || matrix[0].length == 0) {
            return false;
        }
        int rows = matrix.length;
        int cols = matrix[0].length;
        // 右上角数字的下标
        int rowIndex = 0;
        int colIndex = cols-1;
        while(rowIndex < rows && colIndex>=0){
            if(target > matrix[rowIndex][colIndex]){
                rowIndex++;
            }else if(target < matrix[rowIndex][colIndex]){
                colIndex--;
            }else{
                return true;
            }  
            }
        return false;
    }
}

题目5：替换空格

请实现一个函数，把字符串 s 中的每个空格替换成”%20”。

例如：

输入：s = "We are happy."
输出："We%20are%20happy."

使用java api，String类中的函数replace(resouce,target)

class Solution {
    public String replaceSpace(String s) {
        return s.replace(" ","%20");
    }
}

或者：

class Solution {
    public String replaceSpace(String s) {
         StringBuilder sb = new StringBuilder();
         for(int i = 0;i<s.length();i++){
             char c = s.charAt(i);
             if(c==' '){
                 sb.append("%20");
             }else{
               sb.append(c);
             }
         }
         return sb.toString();
     }
}

题目6：从尾到头打印单链表

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

例如：

输入：head = [1,3,2]
输出：[2,3,1]

思路一

从前往后遍历单链表，并存入栈中，遍历完后依次出栈，即为倒序。

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public int[] reversePrint(ListNode head) {
        // 遍历单链表，并入栈，然后出栈，即为逆序
        Stack<Integer> stack = new Stack<>();
        while(head!=null){
            stack.push(head.val);
            head = head.next;
        }
        int[] arr = new int[stack.size()];
        int i = 0;
        while(stack.size()>0){
            arr[i++] = stack.pop();
        }
        return arr;
    }
}

题目9：用两个栈实现队列

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 ）。

思路一

将数字一次压入第一个栈中，然后出栈并压入第二个栈中，第二个栈依次出栈。

class CQueue {

    // 使用LinkedList 作stack的容器
    private LinkedList<Integer> stack1;
    private LinkedList<Integer> stack2;

    public CQueue() {
        stack1 = new LinkedList<>();
        stack2 = new LinkedList<>();
    }

    public void appendTail(int value) {
        stack1.add(value);
    }

    public int deleteHead() {
        if(stack2.isEmpty()){
            if(stack1.isEmpty()){
                return -1;
            }
            while(!stack1.isEmpty()){
                stack2.add(stack1.pop());   
            }
        }
        return stack2.pop();
    }
}
