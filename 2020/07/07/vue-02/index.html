数据的双向绑定

vue.js是ViewModel层的实现者，数据的双向绑定，即当数据发生变化时，视图也就发生变化，当视图发生变化时，数据也会跟着同步变化。

<body>
<div id="app">
    姓名：<input type="text" v-model="message" placeholder="请输入点内容"/>
    <span>输入的内容：{{message}}</span>
    <br/>
    性别：<input type="radio" name="gender" value="1" v-model="gender" >男
         <input type="radio" name="gender" value="0" v-model="gender">女
    <span>选择的是{{gender}}</span>
    <br/>
    学科：<select v-model="major">
        <option value="" disabled>--请选择--</option>
        <option value="java">java</option>
        <option value="python">python</option>
        <option value="c++">c++</option>
    </select>
    <span>已选择：{{major}}</span>
</div>
</body>
<script>
    var v1 = new Vue({
        el:"#app",
        data:{
            message:"Hello,vue!",
            gender: 0,
            major: ""
        }
    });
</script>

结果：

注意：v-model会忽略所有表单元素的value，checked，selected 特性的初始值而总是将Vue实例中的数据作为数据来源。应该通过js在组件的data选项中声明初始值。

组件

组件是vue可复用实例，就是一组模板。

<body>
<div id="app">
    <!-- 传递给组件中的props   -->
    <!-- v-for="item in items" 遍历Vue实例中定义名为items的数组，并创建同等数量的组件 -->
    <!-- v-bind:item="item" 将遍历到的value绑定到组件中prop定义的名为item的属性上
         -->
    <demo v-for="value in items" v-bind:item="value"></demo>
</div>
</body>

<script>
    // 定义一个Vue组件component
    Vue.component("demo",{
        props: ['item'],// 接收组件传来的参数
        template: "<li>{{item}}</li>"
    })

    var v1 = new Vue({
        el:"#app",
        data:{
            items:["java","python","linux"]
        }
    });
</script>

Axios异步通信

Axios是一个可以用在浏览器端和node.js的异步通信框架，主要作用就是实现ajax异步通信。

引入axios.js

<script src="https://unpkg.com/axios/dist/axios.min.js"></script>

创建一个data.json文件作为数据

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>

<body>
    <div id="vue">
        <div>{{info.name}}</div>
        <div>{{info.age}}</div>
        <div>{{info.address}}</div>
        <div>{{info.links}}</div>
        <a v-bind:href="info.links[0].url">点击访问bilibili</a>
    </div>
</body>
<script type="text/javascript">
    var vm = new Vue({
        el: "#vue",
        // data() 接收数据
        data(){
            return{
                // 请求的返回参数必须和json字符串一样
                info:{
                    name: null,
                    age: null,
                    address: {
                        city: null,
                        province: null,
                        country: null
                    },
                    links: [
                        {
                            name: null,
                            url: null
                        },
                        {
                            name: null,
                            url: null
                        },
                        {
                            name: null,
                            url: null
                        }
                    ]
                }
            }
        },
        mounted(){
            axios.get('../data.json').then(response=>(this.info=response.data));
        }
    })
</script>
