vue-cli

vue-cli 就是官方提供的一个脚手架，用于快速生成一个vue的项目模板。

vue-cli安装

1、安装node.js

2、安装node.js淘宝镜像加速器cnpm

# -g全局安装
npm install cnpm -g

3、安装vue-cli

cnpm install vue-cli -g

4、查看可以基于哪些模板创建vue程序

vue list

创建一个vue项目

1、在当前目录创建一个名为myvue的vue项目

vue init webpack myvue

2、将自动安装的选项都选择no，之后进行手动安装

3、进入项目文件夹安装依赖环境

cd myvue
# 根据package.json自动安装所需要的包
npm install

若出现warning，则按照提示去修复

4、修复成功后一个最基础的vue项目就构建完成了

5、项目打包运行

npm run dev

运行成功后可在浏览器端通过 http://localhost:8080进行访问

webpack

webpack是一个js应用程序的静态模块打包器。它会递归地构建一个关系依赖图，其中包含应用程序所需要的各个模块，然后将每个模块打包成一个或多个bundle。将ES6规范的代码打包降级为所有浏览器兼容的ES5。

安装webpack及其客户端

npm install webpack -g

npm install webpack-cli -g

查看其版本号

webpack -v

webpack-cli -v

webpack简单使用

创建hello.js

// 暴露一个方法
exports.sayHi=function () {
    document.write("<h1>Hi</h1>")
}

main.js

var hello = require("./hello");// 引入hello.js
hello.sayHi();// 调用hello.js中的方法

创建webpack打包配置文件webpack.config.js

module.exports = {
    entry: "./modules/main.js",// 入口
    output: {
        filename: "./js/bundle.js"//输出文件
    }
}

创建index.html，引入打包输出的文件，在浏览器查看是否正确

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <script src="dist/js/bundle.js"></script>
</body>
</html>

vue-router


1、在当前项目文件夹中安装vue-router

npm install vue-router --save-dev

2、在当前项目的src文件夹下创建component文件夹用来存放组件

编写所需要的组件
Content.vue

<template>
    <h1>Content.vue</h1>
</template>

<script>
    export default {
        name: "Content"
    }
</script>
<!--scoped  样式仅在当前组件生效-->
<style scoped>
</style>

Main.vue

<template>
    <h1>Main.vue</h1>
</template>

<script>
    export default {
        name: "Main"
    }
</script>

3、在当前项目的src文件夹下创建router文件夹用来存放路由的配置信息

router/index.js

// 导入插件
import Vue from 'vue'
import VueRouter from 'vue-router'
// 导入组件
import Content from "../components/Content"
import Main from "../components/Main"

// 安装路由
Vue.use(VueRouter);

// 配置导出路由
export default new VueRouter({
  routes:[{
    // 路由路径
    path: '/content',
    name: 'content',
    // 跳转的组件
    component:Content
  },
    {
      path: '/main',
      name: 'main',
      component: Main
    }
  ]
});

4、在main.js引用配置的路由

import Vue from 'vue'
import App from './App'
import router from './router'

Vue.config.productionTip = false


new Vue({
  el: '#app',
  // 配置路由
  router,
  components: { App },
  template: '<App/>'
})

5、在App.vue中创建路由的跳转链接

<template>
  <div id="app">
    <h1>App.vue</h1>
    <router-link to="/main">跳转到Main</router-link>
    <router-link to="/content">跳转到Content</router-link>
    <router-view></router-view>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>

<style>
#app {
  text-align: center;
}
</style>

运行项目

npm run dev
